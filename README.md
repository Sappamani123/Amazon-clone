# Amazon Clone UI

A responsive frontend clone of the Amazon homepage built using **HTML5** and **CSS3**. This project replicates the look and feel of Amazon's landing page while focusing on responsive design, clean code, and modern web development practices.

## Features
- Responsive navigation bar
- Amazon-style search bar
- Hero banner section
- Product category cards
- Responsive footer
- Font Awesome icons
- Clean and organized HTML & CSS

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Font Awesome

## Purpose
This project was created for educational purposes to improve frontend development skills by recreating a popular e-commerce website interface. It is not affiliated with or endorsed by Amazon.
